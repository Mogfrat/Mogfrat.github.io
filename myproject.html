Crypto Watcher :

Il permet entre autre de voir l'état de ses workers lors du minage, permet de connaitre les stats des pools, de connaître la valeur "en temps réel" des cryptomonnaies et de connaitre les prix du gas pour les transactions.

API utilisées :

Ethermine, Nanopool, Flexpool, Messari, Polygonscan, Etherscan et Zapper.

Wibot :

Il permet de d'envoyer des images autant SFW que NSFW de waifu. Ainsi que de rechercher après les animés à partir d'une photo et/ou gif.

API utilisées :

Animechan, Anime fact de herokuapp, Waifu.pics, Waifu.im et Trace.moe.
